# PhoneBook-Vue-Webpack

Простая телефонная книга.

### Задание:

Необходимо проект с телефонной книгой "PhoneBook-Vue" настроить с помощью Webpack с нуля. 

Настроить:

- webpack
- babel
- vue components
- bootstrap-vue
- sass
- css minification 

### Сборка и запуск сервера
```
npm run start
```

### Сборка проекта для production
```
npm run build
```
